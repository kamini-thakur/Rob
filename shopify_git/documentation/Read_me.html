Install Theme

To install the theme, please log in to your Shopify store and go to the Themes. From there, click on Upload a Theme and upload the .zip file from the 'upload' folder included with this package

If your store is currently active and is using a different theme, you select button Publish theme what you have just upload

Shop Settings

Currency (General)

Editing the HTML for currency formatting

1. From your shop admin, go to the Settings / General

2. Scroll down and then you will see a "Standards & formats" block. Find the "Currency" there

3. Select your currency

4. Click "change formatting"

5. Replace selected text and click Save

6. Currency must be the same

2. Choose a payment gateway to accept payments for orders. Than checkout page is work

3. Click save

MetizTheme Shopify theme feature with multi-level drop-down menu.To add Sub Menu item to the Main Menu item you have to follow below given steps:

1. Go to Admin Panel Online Store / Navigation

2. Add Main Menu item name from drop-down menu. For example, in below screen shot we had taken Men as a main menu item and then create its sub menu.

3. Now Click "Add menu item" button to create sub menu (like Special Discount, Winter Collection etc)."

4. You can also linked the sub-menu item to any page To add link, refer this https://help.shopify.com/manual/sell-online/online-store/menus-and-links#add-a-link-to-a-menu

Note : Name of the main menu item and handle should be matched.

Filters Overview

You can add filter tags that allows your customer to filter products price, size, color and brand. How to add/remove each of the filter tag is explained seperately.

Add Category Filter Product

Product tags

Filter category work with tags. You need write product category in product tags field.
It work for all products in the collection

Add Color Filter Tags

To add new color in color filter - Go to admin > Products > Products

Write "Color" word with name of the color of your choice. For example, to add Purple color you need to write Color_Purple. To remove any of color just click on cancel button.

Review in Front end

Price Filter

Minimum and maximum price range is set automatically on the basis of products added to collection.

Add Size Filter Tags

To add size in size filter - Go to admin > Products > Products

Write "Size" word with the Size you want to defined. For example, to add XXL size you need to write Size_XXL. To remove any size from filter click on cancel button.

Review in Front end

Add Brand Filter Tags

To add new Brand name in Brand filter - Go to admin > Products > Products

Wirte "Brand"word before the Brand name you want to add. For example, to add "New" as a brand you need to write Brand_New. You can also remove any Brand name from filter by clicking on cancel button.

After that you go to the theme editor. In the field "List Popular Tags" you write all "popular words" in collections

Slideshow settings

Theme offers admin the ability to upload any image (png, jpeg etc) to create slideshow or image slider for your Shopify store.

To add slideshow follow the given below instructions

i. Go to Online Store > Theme > Customize theme > Sections > Add section

ii. Now click on "Add" button to add Slideshow

iii. You will get option to customize the Slideshow. It allows you to enable Auto rotation of slides, its transition time as well as you can add/remove image of your choice.

iv. If you do not want Slideshow as a section you can also remove it. Just click on "Delete section" button.

Featured Collection

Need to show featured or special products on Home page of your store? You can easily do it with this theme with the help of Featured Collection option.

To add Featured collection follow the given below steps:

i. Go to Online Store > Theme > Customize theme > Sections > Add section

ii. Now click on "Add" button to add Featured Collection

iii. You will get many options such as Update, Remove, Edit Collection etc.
Update: This will allow you to upload, edit image and to select image from library.
Remove: To remove the image
Edit Collection: When users click on image, they'll redirect to the collection page defined in the list. For example in a given screenshot, user redirect to Home Page.
